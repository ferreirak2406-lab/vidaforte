# 🌿 VIDA FORTE

Aplicativo de evolução pessoal com hábitos, gamificação e companhia de IA.

## 🎯 Objetivo

Transformar pequenas ações diárias em grandes mudanças de vida através de:
- ✅ Hábitos gamificados
- 🏆 Sistema de XP e níveis
- 🎖️ Conquistas e streaks
- 💬 Mentoria da Dra. Helena
- 📊 Estatísticas e progresso
- 🌍 Comunidade

## 🚀 Como começar

### Instalação
```bash
npm install
```

### Desenvolvimento
```bash
npm start
```

### Build
```bash
npm run build
```

### Deploy no GitHub Pages
```bash
npm run deploy
```

## 📱 Testar no navegador

Acesse: https://ferreirak2406-lab.github.io/vidaforte

## 🔄 Para gerar APK

Você precisa de um PC com:
- Node.js
- npm
- Expo CLI

```bash
npm install -g expo-cli
npm install
npx expo build:android
```

## 📋 Funcionalidades

- 🎮 6 hábitos principais (Água, Treino, Alimentação, Leitura, Sono, Meditação)
- 💾 Sincronização automática com localStorage
- 🌐 Suporte offline-first
- 🌍 Múltiplos idiomas (PT, EN, ES)
- 🎨 Design moderno e responsivo
- 📊 Gráficos de progresso
- 🔐 Backup e restauração de dados
